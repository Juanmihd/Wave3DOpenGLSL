Detailed Description

template<class value_t, class allocator_t = allocator>
class octet::containers::dictionary< value_t, allocator_t >

Map strings to objects and object references.

Strings and objects are owned by the dictionary.

This is like a JavaScript or Python dictionary but for text keys only. It is about twenty times faster than using std::map<std::string, xxx>

Example: 

dictionary<int> my_dict;
my_dict["fred"] = 27; 
my_dict["anne"] = 28; 

int annes_age = my_dict["anne"];
