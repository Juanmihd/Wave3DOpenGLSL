Detailed Description

template<class item_t, class allocator_t = allocator>
class octet::containers::ref< item_t, allocator_t >

The ref class is used to keep reference counter pointers to object.

It should only be used as a data member in a class. Do not use ref as a parameter and only rarely as a stack variable.

In all other cases just use a regular C++ pointer.

Anything using a ref must implement the add_ref() and release() methods. These allow the ref to keep track of the number of uses of an object.

Examples. 

{
  ref<mesh> my_mesh = new mesh(); // create a new mesh.
  ...
  my_mesh->get_num_vertices();    // use ref like a pointer
  mesh *local_ptr = my_mesh;      // convert to an unmanaged pointer
  ...
} // the mesh object will be freed here.
