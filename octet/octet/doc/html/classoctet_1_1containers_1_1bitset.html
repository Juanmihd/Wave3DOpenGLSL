Detailed Description

template<unsigned size_>
class octet::containers::bitset< size_ >

Fixed size bit set class.

Bit sets are useful for keeping track of multiple boolean states. For example, if we have a particle system, we can track the "live" state of a particle by using only one bit per particle.

Example 

bitset<256> is_upper_case;
for (unsigned i = 'A'; i <= 'Z'; ++i) {
  is_upper_case.setbit(i);
}

now we can test if a character is upper case:
if (bitset[chr]) printf("%c is upper case\n", chr);
